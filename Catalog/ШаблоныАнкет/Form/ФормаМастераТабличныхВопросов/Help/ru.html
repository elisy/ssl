Форма мастера табличных вопросов предназначена для редактирования табличных вопросов.

 

На первой странице мастера необходимо выбрать тип табличного вопроса. Табличные вопросы могут быть четырех типов:

 

  • Составной табличный вопрос
  • Табличный вопрос с предопределенными ответами в колонках
  • Табличный вопрос с предопределенными ответами в строках
  • Табличный вопрос с предопределенными ответами в строках и колонках

 

Составной табличный вопрос:

Для составного табличного вопроса задается состав вопросов. Заданные вопросы будут отображаться в качестве заголовков колонок таблицы. Респондент, отвечающий на вопросы анкеты, будет иметь возможность добавить произвольное количество строк  в таблицу при ответе.

 

Табличный вопрос с предопределенными ответами в строках:

Для данного типа табличного вопроса создается состав вопросов. Заданные вопросы будут отображаться в качестве заголовков колонок таблицы. Для первого вопроса, составитель анкеты должен будет задать предопределенные ответы. Данные предопределенные ответы будут отображаться в строках первой колонки таблицы. При ответе, респондент не сможет добавить или удалить строки из таблицы. Количество строк таблицы будет равно количеству предопределенных ответов.

 

Табличный вопрос с предопределенными ответами в колонках:

Для данного типа табличного вопроса создается состав вопросов. Заданные вопросы будут в строках первой колонки таблицы. Для первого вопроса, составитель анкеты должен будет задать предопределенные ответы. Данные предопределенные ответы будут отображаться в качестве заголовков колонок таблицы. При ответе, респондент не сможет добавить или удалить строки из таблицы. Количество строк таблицы будет равно количеству вопросов, уменьшенному на один.

 

Табличный вопрос с предопределенными ответами в строках и колонках:

Для данного типа табличного вопроса задается три вопроса. Для первых двух вопросов составитель анкеты должен будет задать предопределенные ответы. Предопределенные ответы на первый вопрос будут расположены в строках первой колонки. Предопределенные ответы на второй вопрос будут отражать в качестве заголовков колонок. Ответы на третий вопрос пользователь и должен будет указывать в ячейках таблицы. При ответе, респондент не сможет добавить или удалить строки из таблицы. Количество строк таблицы будет равно количеству предопределенных ответов на первый вопрос.
