Справочник "Внутренние документы" служит для учета официальных информационно-справочных или организационно-распорядительных документов.

К каждому внутреннему документу можно присоединить один или несколько файлов. Учет файлов возможен, если настройка программы "Использовать файлы у внутренних документов" установлена.

Для документа следует задать: 

  • папку внутренних документов, в которой будет размещен документ;
  • пользователя, подготовившего документ (при создании нового документа заполняется программой по текущему пользователю);
  • подразделение, которое выпускает документ;
  • руководителя (пользователя), утвердившего (подписавшего) документ;
  • корреспондента, с которым связан документ, показывается, если для указанного вида документа в настройках установлено "Вести учет по корреспондентам";
  • контактное лицо выбранного корреспондента, показывается, если для указанного вида документа в настройках установлено "Вести учет по корреспондентам";
  • срок исполнения - задается дата исполнения документа, показывается, если для указанного вида документа в настройках установлено "Использовать срок исполнения";
  • гриф доступа - если установлена настройка "Использовать грифы доступа", то внутреннему документу нужно обязательно присвоить гриф доступа, выбрав его из числа заранее настроенных администратором грифов. Гриф определяет права доступа пользователей к внутреннему документу;
  • вид документа - показывается, если настройка программы "Использовать виды внутренних документов" установлена;
  • состояние документа - показывается, если настройка программы "Использовать состояние документов" установлена;
  • номенклатура дел - показывается, если для указанного вида документа в настройках установлено "Вести учет по номенклатуре дел";
  • организация - в случае ведения документооборота для нескольких организаций (показывается, если настройка программы "Использовать учет по организациям" установлена);


Документ может нумероваться вручную или автоматически (в зависимости от настроек вида документа).
Если задана автоматическая нумерация, то для того, чтобы присвоить регистрационный номер необходимо нажать кнопку "Зарегистрировать".
Если задана ручная нумерация, то достаточно заполнить поля "Регистрационный номер" и "Дата регистрации" и записать документ.

Если настройка программы "Использовать электронно цифровые подписи" включена, то в карточке документа добавляются группа команд "ЭЦП" (содержит команду "Подписать", с помощью которой выполняется удостоверение документа электронной подписью) и закладка "ЭЦП", в которой редактируются и проверяются электронно цифровые подписи.
Если документ подписан, то его редактирование невозможно. Для редактирования нужно сначала удалить все подписи. Текущий пользователь может удалять только свои подписи. Право удаления всех подписей принадлежит администратору.
Подпись можно сохранить в файл (например, для пересылки по электронной почте), а также посмотреть сертификат подписи.
