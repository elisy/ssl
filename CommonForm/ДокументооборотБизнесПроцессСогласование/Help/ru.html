Этот бизнес-процесс обходит всех участников согласования (рецензентов) по заранее составленному списку, после чего возвращается к автору (инициатору бизнес-процесса). В случае отрицательного результата согласования автор имеет возможность повторить согласование.

При заполнении списка рецензентов можно указывать как непосредственно исполнителя (пользователя), так и роли.

Предусмотрено три варианта обхода согласующих - "все вместе", "по очереди" и "смешанно". Если установлен тип согласования "все вместе", то задачи всем рецензентам формируется одновременно. При этом бизнес-процесс возвращается к автору после завершения задач всеми согласующими.  Если установлен тип согласования "по очереди", то задача следующему рецензенту формируется только после завершения задачи предыдущему. При этом бизнес-процесс сразу возвращается к автору, как только один из согласующих принял отрицательное решение. Если установлен тип согласования "смешанно", то для каждого исполнителя можно указать порядок формирования задачи - вместе с предыдущим либо после предыдущего. В результате тип согласования "смешанно" позволяет сочетать этапы одновременного и последовательного согласования.

Каждому из исполнителей на выполнение задачи отводится срок, указанный в карточке бизнес-процесса.  При указании срока следует учитывать, что для варианта согласования "все вместе" ожидаемый срок завершения бизнес-процесса совпадает со сроком, указанным в карточке. Тогда как для варианта согласования "по очереди" ожидаемый срок завершения бизнес-процесса равен произведению количества исполнителей и срока, отведенного каждому исполнителю.

Этот бизнес-процесс можно создавать непосредственно из карточки документа при помощи меню "Создать бизнес-процесс".

Для просмотра текущего состояния бизнес-процесса (списка выполненных и не выполненных задач) используйте команду "Задачи" в панели навигации карточки документа, отправленного на согласование.

Если не был выполнен запуск бизнес-процесса, заполнение формы может быть выполнено на основании шаблона бизнес-процесса. Для этого необходимо выполнить команду "Заполнить по шаблону" на командной панели формы. Если для данного типа бизнес-процесса и вида предмета бизнес-процесса настроены шаблоны бизнес-процессов, то система предложит выбрать один из шаблонов для заполнения формы. Если был настроен только один шаблон, то заполнение формы произойдет автоматически.
