Этот бизнес-процесс используется для автоматизации исполнения распоряжений руководителя одним или несколькими исполнителями с одновременным контролем.

Если указано несколько исполнителей, то одного из них можно указать ответственным исполнителем. При старте этого бизнес-процесса соответствующие задачи на исполнение автоматически получат все исполнители по списку. Из задачи ответственного исполнителя имеется возможность просмотреть задачи остальных исполнителей.

Пользователь, указанный в поле "Контролер", получит задачу Контролировать одновременно с исполнителями. В обязанности контролера обычно входит слежение за сроками исполнения.

Пользователь, указанный в поле "Проверяющий", получит задачу после того, как все исполнители выполнили свои задачи. В обязанности проверяющего входит проверка результата исполнения задач. Проверяющий может возвращать задачи исполнителей на доработку при помощи соответствующей команды в карточке своей задачи.

Этот бизнес-процесс можно создавать непосредственно из карточки, например, входящего документа, при помощи меню "Создать бизнес-процесс".

Если не был выполнен запуск бизнес-процесса, заполнение формы может быть выполнено на основании шаблона бизнес-процесса. Для этого необходимо выполнить команду "Заполнить по шаблону" на командной панели формы. Если для данного типа бизнес-процесса и вида предмета бизнес-процесса настроены шаблоны бизнес-процессов, то система предложит выбрать один из шаблонов для заполнения формы. Если был настроен только один шаблон, то заполнение формы произойдет автоматически.
