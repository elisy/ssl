// Процедура ПередПроверкойЗапретаИзменения позволяет по некоторому
// условию, например, для отказа проверки в некоторых режимах работы
// подсистемы обмена данными (если она используется) или
// для отказа от проверки по статусу документа (открыт/закрыт).
//
// Параметры:
//  Объект       - объект или набор записей (ПередЗаписью или ПриЧтенииНаСервере):
//                 СправочникОбъект
//                 ДокументОбъект
//                 ПланВидовХарактеристикОбъект
//                 ПланСчетовОбъект
//                 ПланВидовРасчетаОбъект
//                 БизнесПроцессОбъект
//                 ЗадачаОбъект
//                 ПланОбменаОбъект
//                 РегистрСведенийНаборЗаписей
//                 РегистрНакопленияНаборЗаписей
//                 РегистрБухгалтерииНаборЗаписей
//                 РегистрРасчетаНаборЗаписей
//
//  СтандартнаяОбработка - Булево, начальное значение Истина,
//                 если присвоить значение Ложь,
//                 тогда стандартная проверка выполняться не будет.
//
//  УзелПланаОбмена - Неопределено, ПланыОбменаСсылка.<Имя плана обмена> -
//                 начальное значение Неопределено, если присвоить конкретную ссылку,
//                 тогда будет выполнятся проверка запрета загрузки.
//
//  СообщитьОЗапрете - Булево, начальное значение Истина. Если задать Ложь,
//                 тогда пользователь не увидит сообщение о запрете, а
//                 увидит лишь отказ записи.
//                 В журнал регистрации сообщение будет записано в любом случае.
//
Процедура ПередПроверкойЗапретаИзменения(Объект, СтандартнаяОбработка, УзелПланаОбмена, СообщитьОЗапрете) Экспорт
