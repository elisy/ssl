Адресный классификатор предназначен для поддержки единого способа получения, использования и поддержки адресных сведений в прикладных подсистемах.  Механизм предоставляет инструменты для управления адресными сведениями: добавления, изменения, удаления адресных сведений, в том числе загрузка адресных сведений с диска информационно-технологического сопровождения либо файлов распространяемых ГНИВЦ ФНС РФ.

Информация по адресным сведениям хранимая в информационной базе представлена в виде таблицы со следующими колонками:

Тип Тип адресного объекта, в зависимости от уровня классификации; поле может принимать значение от 1 до 6
Код Уникальный код адресного объекта (составное поле)
СС Код субъекта Российской Федерации
РРР Код района
ГГГ Код города
ППП Код населенного пункта
УУУ Код улицы
Наименование Наименование адресного объекта
Сокр. Сокращенное наименование типа адресного объекта
Индекс Содержит почтовый индекс предприятия почтовой связи, обслуживающего данный адресный объект

Альтернативные названия

 Альтернативное наименование адресного объекта (например устаревшее название)

 

Командная панель формы содержит следующие кнопки для управления адресными сведениями:

  • Загрузить - вызов мастера загрузки адресных сведений (открывается форма подготовки параметров загрузки)
  • Очистить классификатор - вызов формы управления очисткой адресных сведений по регионам

    • Адресными сведениями можно так же управлять непосредственно добавляя / удаляя / изменяя строки в табличной части.

    Изначально адресный классификатор содержит заполненные сведения по адресным объектам первого уровня классификации .
